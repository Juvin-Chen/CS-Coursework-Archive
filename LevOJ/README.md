# LevOJ 课程作业汇总

本目录用于存放 **程序设计基础、程序设计实践、算法设计与分析** 三门课程在 LevOJ 算法平台上的作业代码。

## 包含课程

| 课程名称       | 对应内容       |
| :------------- | :------------- |
| 程序设计基础   | 入门级编程题目 |
| 程序设计实践   | 综合编程训练   |
| 算法设计与分析 | 经典算法题目   |

## 目录结构

```
LevOJ/
├── 程序设计基础/    # 对应课程的作业代码
├── 程序设计实践/    # 对应课程的综合编程作业
└── 算法设计与分析/  # 经典算法（如排序、搜索、动态规划等）解题代码
```

## 使用说明

- 代码仅作学习参考，建议先独立思考解题，非专业算法选手，只是个人解法；
- 若涉及算法优化，会在代码中注明核心思想与时间 / 空间复杂度。
