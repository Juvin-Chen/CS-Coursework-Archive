# EvoBrawl

基于 EasyX 图形库的 **双人进化对战小游戏**

## 游戏简介

控制小兔子在场景内收集饮料，成长进化并释放光波技能，击败对手即可获胜。

游戏支持双人同屏对战，自动记录每局胜负帧数，可回溯成绩。

## 操作说明

- 玩家 1
  - 移动：↑ ↓ ← → 方向键
  - 光波技能：回车键
- 玩家 2
  - 移动：W A S D
  - 光波技能：F 键
- 退出：ESC

## 成长与技能

- 收集 ≥20 瓶饮料：兔子变大、移动速度加快
- 收集 ≥30 瓶饮料：解锁**光波技能**
- 光波命中对方 → 游戏胜利

## 开发环境

- 系统：Windows
- IDE：Visual Studio 2022
- 图形库：EasyX
- 语言：C++14

## 运行与记录

1. 配置好 EasyX 后直接编译运行
2. 游戏结束自动显示胜利画面
3. 每局对战帧数自动保存到 `best_kos.txt`

## 核心特性

- 900×900 画布，双缓冲流畅渲染
- 精准距离 & 直线碰撞判定
- 动态成长系统
- 对战数据本地持久化
