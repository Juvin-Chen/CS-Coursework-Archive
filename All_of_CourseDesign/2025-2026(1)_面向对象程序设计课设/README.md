## 项目简介

Campus RainHub 是一款面向校园场景的共享雨伞服务平台，旨在解决高校师生突遇降雨时的出行痛点。项目基于**面向对象设计思想**，采用「共享充电宝」模式实现校园雨具的 24 小时无人值守借还，涵盖用户自助终端与运营管理后台两大核心应用，是一套集智能终端交互、实时状态监控、数据可视化于一体的物联网解决方案。

## 技术栈

C++17  /  Qt 6.9.3（Qt Widgets + QSS 界面美化）/  MySQL 8.0  /  CMake 3.16+

## 核心功能

### 1. 用户自助终端（Client）

- 多类型雨具管理：支持普通塑料伞、抗风伞、遮阳伞、雨衣等品类（工厂模式实现）
- 可视化校园地图：实时渲染站点位置、剩余库存（可用 / 空槽）
- 一卡通交互：模拟校园卡借还流程，内置使用指南与收费说明

### 2. 综合管理后台（Admin）

- 全局站点监控：实时查看所有终端在线状态、库存水位
- 生命周期管理：站点 / 雨具状态变更、用户账号激活 / 冻结
- 安全运维：管理员权限密码重置、订单流水可视化统计
